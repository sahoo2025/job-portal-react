# 🚀 React Job Portal

A fully responsive Job Listing Web Application built using **React.js** with real-time API integration.  
Users can search, filter, and browse job listings with pagination support.

## 📌 Project Overview

React Job Portal is a dynamic web application that fetches real-time job listings from an external API and displays them in a clean, user-friendly interface.

The application allows users to:

- 🔎 Search jobs by title
- 📍 Filter jobs by location
- 📄 Navigate using pagination
- 🌐 View real job postings via external links
- 📱 Access the platform on mobile and desktop (Responsive Design)
